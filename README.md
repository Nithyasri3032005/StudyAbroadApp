# Study Abroad App

A modern React Native application for exploring universities worldwide.

## Features

- Search universities
- University details page
- Favorites feature
- Dark mode support
- Google Maps integration
- Official website redirection

## Installation

```bash
git clone https://github.com/Nithyasri3032005/StudyAbroadApp.git

cd StudyAbroadApp

npm install

npx react-native run-android
```
